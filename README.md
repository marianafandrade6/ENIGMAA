🕵️ Guia de Uso: O Decifrador
Bem-vindo ao O Decifrador! Esta ferramenta interativa permite que você transforme mensagens de texto usando dois métodos diferentes. Siga este guia para aprender a codificar e decifrar suas mensagens.

🛠️ Modos de Operação
A interface é dividida em duas abas principais no topo:

Cifra de César: Um método de criptografia antigo que substitui cada letra por outra no alfabeto.

Base64: Um método de codificação moderno usado para transportar dados (não é considerado criptografia de segurança, mas sim formatação).

🔑 1. Usando a Cifra de César
Este modo é perfeito para criar mensagens secretas baseadas em um "pulo" no alfabeto.

Escolha a Ação: Clique em Criptografar (para esconder a mensagem) ou Descriptografar (para ler uma mensagem cifrada).

Chave de Deslocamento: Use o controle deslizante (slider) para escolher um número de 1 a 25. Se a chave for 3, o 'A' vira 'D', o 'B' vira 'E', e assim por diante.

Visualização do Alfabeto: O quadro "Mapeamento do Alfabeto" mostra exatamente qual letra substituirá cada caractere em tempo real.

Entrada de Texto: Digite sua mensagem em "Texto Original".

Resultado: A mensagem cifrada aparecerá no quadro inferior. Você pode usar o botão Copiar para enviá-la a alguém!

Dica: Para descriptografar, você precisa saber qual foi a "Chave" usada originalmente. Se recebeu uma mensagem com chave 3, coloque o slider no 3 e selecione "Descriptografar".

📑 2. Usando o Base64
O Base64 transforma seu texto em uma sequência de caracteres que pode ser lida por qualquer sistema de computador.

Escolha a Ação: Clique em Codificar ou Decodificar.

Processo: Ao contrário da Cifra de César, o Base64 não usa chave. Ele apenas converte o texto para um formato binário e depois para uma tabela de 64 caracteres.

Entrada de Texto: Digite seu texto em "Texto Original".

Resultado: O código gerado geralmente termina com sinais de igual (=), que são marcas de preenchimento do padrão Base64.

⚡ Atalhos Rápidos
Botão de Inversão (⇅): No centro da tela, este botão troca o conteúdo do campo de entrada com o de saída, facilitando o teste reverso.

Limpar (↺): O ícone de seta circular no canto superior do campo de texto limpa o que você digitou para começar do zero.

Copiar: Localizado no campo de saída, envia o resultado direto para sua área de transferência.
